# 진실의 옥상 (Truth Rooftop) 🌙

> "야, 잠깐 옥상으로 올라올래?"
>
> 친구들과 특별한 장소에서 만나는 감성 장소형 메신저 🌙🏪✨

## 프로젝트 개요

진실의 옥상은 단순한 메신저가 아니다. 친구들과 대화하다가 특정 장소로 초대하여 특별한 분위기에서 대화를 나눌 수 있는 **"장소 기반 메신저"**다.

일반 채팅과 별개로 장소마다 다른 분위기와 연출을 제공하며, 친구들끼리 장난스럽게 놀거나 평소와 다른 이야기를 나눌 수 있는 공간을 만드는 것이 목적이다.

초기에는 "답장을 안 보면 옥상으로 끌려가는 메신저" 컨셉이었지만, 발전 과정에서 **"친구끼리 특별한 장소로 초대하는 이벤트 메신저"** 방향으로 변경되었다.

## 핵심 컨셉

### 1. 기본 채팅

일반적인 메신저처럼 사용한다.

- 친구 추가
- 그룹채팅
- DM
- 프로필
- 이모티콘

### 2. 장소 초대 시스템

채팅 중 친구에게 초대장을 보낼 수 있다.

| 초대장 종류 |
| --- |
| 옥상 초대장 |
| 편의점 초대장 |
| 공원 벤치 초대장 |
| PC방 초대장 |
| 카페 초대장 |
| 버스정류장 초대장 |

상대방은 **수락하기** 또는 **회피하기**를 선택할 수 있다.

#### 옥상 초대 과정 예시

```
현채가 달걀에게 "옥상 초대장" 발송
        ↓
[달걀 화면] 누군가 옥상 문을 두드리고 있습니다.
        [문 열기]  [회피하기]
        ↓
      문 열기
        ↓
   옥상 초대장 확인
        ↓
       수락
        ↓
     옥상 입장
```

### 3. 회피하기 시스템

단순한 "무시하기"가 아니라 **"회피하기"**라는 개념이다. 창이 그냥 닫히는 게 아니라, 캐릭터가 회피 행동을 한다.

| 장소 | 회피 연출 | 설명 |
| --- | --- | --- |
| 옥상 | 문고리를 잡았다가 놓음 / 교실 천장에 올라감 | "옥상은 부담스러워서 천장으로 피신했습니다." |
| 편의점 | 냉장고 뒤에 숨음 | "아이스크림 구경하는 척합니다." |
| 버스정류장 | 방금 도착한 버스를 타고 사라짐 | "다음 정류장에서 내릴 예정입니다." |
| 카페 | 메뉴판 뒤에 숨음 | — |
| PC방 | 게임 시작 버튼 누름 | "랭크전 중입니다." |

## 장소 시스템

이 앱의 핵심은 **장소**다. 장소마다 분위기와 기능이 다르다.

### 🌙 옥상

가장 특별한 공간. 밤하늘, 도시 야경, 노을, 별빛, 진솔한 대화.
분위기: "야 잠깐 올라와봐" 느낌.

### 🏪 편의점 앞 의자

친구들과 편의점에서 컵라면 먹으며 떠드는 분위기.
주제: 수다 · 장난 · 밈 · 아무말 대잔치. (예: "크르르르갉")

### 🌳 공원 벤치

조용한 대화. 고민상담 · 근황토크 · 소수인원.

### 🚏 버스정류장

짧은 대화 (3~5분 제한). "잠깐 이야기" — 스쳐가는 인연.

### 🎮 PC방

게임 중심. 게임 모집 · 랭크 이야기 · 스크림 모집.

### ☕ 카페

1:1 대화. 잔잔한 분위기.

### 장소 입장 방식 — 초대 코드

셋로그 느낌. 특정 코드가 있어야 입장 가능하다. 비밀 아지트 느낌.

```
ROOM-YACHA-001
```

코드를 가진 사람만 입장할 수 있다.

## 캐릭터 시스템

사용자는 개인 캐릭터를 가진다. 프로필 대신 **캐릭터 중심**이다.

### 장소별 행동

| 장소 | 행동 |
| --- | --- |
| 옥상 | 난간 기대기 · 별 보기 · 앉아 있기 |
| 편의점 | 컵라면 먹기 · 음료 마시기 · 의자 흔들기 |
| 공원 | 벤치에 눕기 · 새 구경하기 |
| PC방 | 키보드 치기 · 게임하기 |

| 장소 | 초대 연출 |
| --- | --- |
| 옥상 | 문 두드리기 / 종이비행기 날리기 |
| 편의점 | 음료수 캔 굴리기 |
| 공원 | 벤치 두드리기 |
| PC방 | 게임 초대 알림 |

## 특별 채팅방

초대를 수락하면 일반 채팅방이 아닌 **특별 장소 채팅방**으로 이동한다.

- 5분 옥상 채팅방
- 10분 편의점 채팅방
- 30분 PC방 채팅방

### 종료 시 문구 예시

- 🌙 옥상은 다시 조용해졌습니다.
- 🌳 공원에 다시 바람이 붑니다.
- 🏪 편의점 의자가 비었습니다.
